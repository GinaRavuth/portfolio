it 3132 web software group project


project idea

In IT 3132 Web Software, there is a final group project assigned.

what it includes

Since there is a checkout system, an administrator dashboard will also be apart of this project. This dashboard will only be accessed by administrators and will allow them to approve checkouts and returns made by students.

my role

For this project, each member of the group has a role. My roles include Front-End Developer, Designer, and Project Manager for Ordino. As Front-End Developer, I worked with HTML/HTML5 + CSS and the Bootstrap framework to develop the front end aspect of the system. As Designer, I wireframed how the system was going to look and tested out colors to see what would fit. Last, as Project Manager, I made sure our group had everything together, reminded us of deadlines, and made sure we were on task.
